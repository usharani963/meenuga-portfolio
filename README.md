### ✅ Use Your Preferred IDE Locally

> **Prerequisite**: Make sure you have Node.js and npm installed.  
> [Install Node.js with NVM](https://github.com/nvm-sh/nvm#installing-and-updating)

```bash
# 1. Clone the repository
git clone https://github.com/usharani963/meenuga-portfolio.git

# 2. Navigate into the project directory
cd meenuga-portfolio

# 3. Install dependencies
npm install

# 4. Start the development server
npm run dev
